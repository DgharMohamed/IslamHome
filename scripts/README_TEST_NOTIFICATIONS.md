# 🧪 سكريبت اختبار أنظمة الإشعارات

سكريبت شامل لاختبار جميع أنظمة الإشعارات في التطبيق تلقائياً.

## 📋 المتطلبات

- Dart SDK مثبت على النظام
- مشروع Flutter مع ملفات الخدمات

## 🚀 الاستخدام

### تشغيل أساسي
```bash
dart scripts/test_notifications.dart
```

### تشغيل مع تقرير HTML
```bash
dart scripts/test_notifications.dart --html
```

### تشغيل مع تقرير JSON
```bash
dart scripts/test_notifications.dart --json
```

### تشغيل مع جميع التقارير
```bash

```

## 📊 التقارير

### تقرير نصي (افتراضي)
يتم إنشاء ملف نصي تلقائياً:
```

```

### تقرير HTML (اختياري)
تقرير تفاعلي بتصميم جميل:
```

```

### تقرير JSON (اختياري)
تقرير بصيغة JSON للمعالجة البرمجية:
```

```

## 🔧 التخصيص

يمكنك تعديل السكريبت لإضافة اختبارات جديدة:

```dart
// إضافة اختبار جديد
await runTest(
  'اسم الاختبار',
  () async {
    // كود الاختبار
    if (condition) {
      throw Exception('رسالة الخطأ');
    }
  }
);
```

## 🐛 استكشاف الأخطاء

### الخطأ: ملف غير موجود
تأكد من وجود ملفات الخدمات في المسارات الصحيحة:
- `lib/services/notification_service.dart`
- `lib/services/quran_audio_handler.dart`
- `lib/services/download_service.dart`

### الخطأ: Dart غير مثبت
قم بتثبيت Dart SDK:
```bash
# على macOS
brew install dart

# على Linux
sudo apt-get install dart

# على Windows
choco install dart-sdk
```
